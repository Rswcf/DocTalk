This folder is reserved for demo PDF files used by the landing page demo.

Expected filenames (place your own sample PDFs here):

- 10k-sample.pdf
- paper-sample.pdf
- contract-sample.pdf

Notes:

- Files in this directory are publicly served by Next.js at `/samples/<filename>`.
- In local dev, the demo page will render these via the PdfViewer.
- If these files are missing, the viewer will show a loading or error state.
